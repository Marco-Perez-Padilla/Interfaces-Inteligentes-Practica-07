# Interfaces-Inteligentes-Practica-07

En esta práctica se experimenta con Vuforia. Se crea una base de datos con dos imágenes, cada una de ellas con un objeto hijo (un escudo). Cuando se enfoca con la cámara la imagen correspondiente, se invoca al escudo de la imagen y se lanza un evento por el cual el escudo cambia su color
